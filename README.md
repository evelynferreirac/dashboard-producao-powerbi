# 📊 Dashboard de Produção com Power BI

## 📌 Sobre o projeto
Este projeto consiste em um dashboard interativo desenvolvido no Power BI para análise de dados de produção, com foco em monitoramento de desempenho, qualidade e produtividade operacional.

## 🎯 Objetivo
Fornecer uma visão clara e estratégica dos indicadores de produção, permitindo identificar gargalos, otimizar processos e apoiar a tomada de decisão baseada em dados.

## 🛠️ Tecnologias e ferramentas
- Power BI  
- Excel  
- Modelagem de Dados  
- ETL (Extração, Transformação e Carga de dados)  
- DAX (criação de medidas e cálculos)

## 📊 Indicadores analisados (KPIs)
- Total aprovado  
- Total rejeitado  
- Horas produtivas  
- Horas paradas  
- Disponibilidade (%)  
- Qualidade (%)  

## 📈 Análises realizadas
- Produção mensal (evolução ao longo do tempo)  
- Comparativo entre produção aprovada e rejeitada  
- Análise de produtividade  
- Monitoramento de desempenho operacional  

## 🧠 Insights gerados
- Identificação de períodos com queda de produção  
- Relação entre horas paradas e produtividade  
- Avaliação da qualidade do processo produtivo  
- Apoio na identificação de oportunidades de melhoria
